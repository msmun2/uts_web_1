# Curriculum Vitae Website

## Deskripsi Project

Project ini merupakan website **Curriculum Vitae (CV) Online** yang dibuat untuk memenuhi tugas **Ujian Tengah Semester (UTS) Mata Kuliah Pemrograman Web I** Program Studi Informatika Universitas Siber Asia.

Website ini dibuat menggunakan **HTML5**, **CSS**, dan **Bootstrap 5** dengan tampilan modern, responsif, dan mudah diakses melalui browser.

---

## Fitur Website

Website ini memiliki beberapa fitur utama, yaitu:

1. Menampilkan foto profil.
2. Menampilkan deskripsi diri.
3. Menampilkan riwayat pendidikan.
4. Menampilkan pengalaman kerja.
5. Menampilkan daftar keahlian.
6. Menampilkan hobi dan minat.
7. Menampilkan galeri dokumentasi.
8. Menampilkan hyperlink ke website eksternal.
9. Responsive pada perangkat desktop maupun mobile.

---

## Komponen HTML yang Digunakan

### Heading

Digunakan untuk membuat judul dan subjudul halaman.

```html
<h1>Curriculum Vitae</h1>
<h2>Tentang Saya</h2>
```

### Komentar HTML

Digunakan untuk menyimpan identitas mahasiswa pada source code.

```html
<!--
Nama :
NIM :
Kelas :
Mata Kuliah :
-->
```

### Text Formatting

Digunakan untuk memperjelas informasi penting pada CV.

```html
<b>Bold</b>
<i>Italic</i>
<u>Underline</u>
<mark>Highlight</mark>
```

### Image

Digunakan untuk menampilkan foto profil dan dokumentasi kegiatan.

```html
<img src="images/foto.jpg">
```

### Hyperlink

Digunakan untuk menghubungkan website dengan halaman lain.

```html
<a href="https://github.com">GitHub</a>
```

### List

Digunakan untuk menampilkan keahlian dan hobi.

```html
<ul>
<li>HTML</li>
</ul>

<ol>
<li>Membaca</li>
</ol>
```

### Table

Digunakan untuk menampilkan data pendidikan dan pengalaman kerja.

```html
<table>
<tr>
<td>Data</td>
</tr>
</table>
```

## Cara Menjalankan Project

1. Download atau clone repository ini.
2. Pastikan seluruh file gambar berada pada folder `images`.
3. Buka file `index.html` menggunakan browser.
4. Website siap digunakan.
